# cine-mania-microsservi-os
CineMania é um ecossistema de microsserviços Java/Spring Boot para venda de ingressos sob alta concorrência. Usa bancos isolados via Docker, cache e Distributed Locks com Redis para evitar vendas duplicadas. Focado na Tríade CIA e na LGPD, traz segurança com OAuth2/JWT, além de resiliência distribuída usando o ecossistema Spring Cloud.
